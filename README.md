### Setup:

clone the project into your plugin folder:

```shell
git clone https://github.com/erikyo/typescript-wp-block.git
```
Then Replace in the boilerplate folder every occurrence of word "boilerplate" with your own plugin name
install with npm.

```shell
npm install
```
